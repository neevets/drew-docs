# Privacy Policy

**Last Updated:** January 2026

This Privacy Policy explains what data Drew collects, how it is used, and your rights regarding that data. By using Drew, you agree to this policy.

---

## Data We Collect

Drew only collects data necessary for its functionality, including:

- Discord User ID  
- Server-related configuration data  
- Feature-related statistics and activity timestamps required for bot functionality  

No additional personal data is collected.

---

## What We Never Collect

Drew does **not** collect or store:

- Message content, chat logs, or conversations  
- Personal or sensitive information  
- Usernames, avatars, or profile details beyond what Discord temporarily provides through its API  

---

## How We Use Data

Collected data is used only to:

- Provide and maintain bot features  
- Store user or server settings  
- Generate statistics, leaderboards, or feature-related information  
- Improve bot stability and performance  

Your data is never sold, shared, or transmitted to third parties.

---

## Data Security

- Data is stored in a secure database  
- Access is limited to systems required for bot operation  
- Reasonable security measures are applied to protect stored data

---

By continuing to use Drew, you acknowledge that you have read and accepted this Privacy Policy.
