# /log

Logging command group.

## Syntax

!!!info Quick Syntax
**Syntax:** `/log <subcommand> [options]`  
**Example:** `/log set #logs`


## Subcommands

- [/log set](set.md)
- [/log clear](clear.md)

Configure logging before enabling aggressive security modules.
