## What Drew Does

- Prevents spam, raids, and malicious automation abuse
- Mitigates destructive actions through anti-nuke controls and recovery options
- Supports daily moderation operations for staff teams
- Logs relevant events for audit visibility
- Handles member verification with interactive panels
- Provides AV utilities for suspicious artifacts and links

## Technical Stack

- Language: Python
- Database: PostgreSQL
- Cache: Redis
