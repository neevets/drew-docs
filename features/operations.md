# Operations

Drew is designed for controllable, guild-level operations.

## Command Interfaces

- Slash commands for guided configuration
- Prefix commands for fast operator workflows

## Configuration Model

- Per-guild toggles for OPSEC modules
- Tunable anti-spam thresholds
- Tunable anti-raid thresholds
- Configurable response and punishment policies

## Access Requirement

- `Administrator` permission is required for full operational coverage.

## Operational Benefits

- Faster incident response
- Reduced moderator overhead
- Consistent enforcement across channels
