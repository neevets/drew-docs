# Security

Drew provides layered OPSEC controls for proactive and reactive server defense.

## Moderation

- Advanced moderation: ban, kick, timeout, purge
- Anti-nuke enforcement with executor timeouts and auto recovery
- Raid mode lockdown with automatic slowmode

## Automated Protection

- Automod with link and invite filtering
- Anti-spam with configurable thresholds
- Anti-raid with join burst detection
- Account age gating and quarantine timeouts

## High-Alert Controls

- Beast Mode for hardened security posture
- Whitelist support for trusted users and roles
- Event logging for critical security actions
- Verification workflow with interactive panel
