# OPSEC Commands

Use these commands to manage OPSEC modules and enforcement behavior.

## Core Management

- `/opsec status`
- `/opsec enable`
- `/opsec disable`
- `/opsec init`

## Spam Controls

- `/opsec spam-config`
- `/opsec spam-punishment`

## Raid Controls

- `/opsec raid-config`
- `/opsec raid-response`
- `/opsec beast-mode`

## Automod and Anti-Nuke

- `/opsec automod-sync`
- `/opsec nuke-config`
- `/opsec log-debounce`

## Whitelist Management

- `/opsec whitelist-add-user`
- `/opsec whitelist-remove-user`
- `/opsec whitelist-add-role`
- `/opsec whitelist-remove-role`
- `/opsec whitelist-list`
