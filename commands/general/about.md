# About

**Drew** is a Discord automation and utility service for structured community operations.

!!!primary Product Intent
Drew is designed for reliable, policy-aligned server operations with minimal configuration friction.

## Design Priorities

Drew is built around:

- Reliability in day-to-day usage
- Predictable command behavior
- Minimal and responsible data handling

## Documentation Principles

This documentation follows a production-first approach:

- Clear prerequisites before deployment steps
- Explicit validation checkpoints after installation
- Troubleshooting guidance ordered by highest-likelihood causes

For legal and data processing details, please review the `legal` section of this documentation.
