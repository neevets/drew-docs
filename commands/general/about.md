# About

**Drew** is an advanced OPSEC Discord bot designed for security-first server management.

!!!primary Product Intent
Drew is built to enforce strong moderation controls, automate incident response, and reduce operational risk in Discord communities.

## Design Priorities

Drew is built around:

- Security-by-default moderation behavior
- Predictable and auditable command execution
- Fast containment during high-risk events
- Minimal and responsible data handling

## Functional Scope

- OPSEC module management per guild
- Anti-spam and anti-raid controls
- Anti-nuke enforcement and executor handling
- Verification and logging channels
- Slash commands and prefix command compatibility

For legal and data processing details, please review the `legal` section of this documentation.
