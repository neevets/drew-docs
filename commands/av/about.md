# /av

Command group for suspicious file and URL triage.

## Syntax

!!!info Quick Syntax
**Syntax:** `/av <subcommand> [options]`  
**Example:** `/av scanfile malware_sample.zip`


## Subcommands

- [/av scanfile](scanfile.md)
- [/av scanweb](scanweb.md)
- [/av rescanfile](rescanfile.md)
- [/av filerelations](filerelations.md)
- [/av scanhistory](scanhistory.md)
- [/av flags](flags.md)

Use this group for rapid analysis and indicator validation before moderation or incident escalation.
