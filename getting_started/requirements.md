# Requirements

Before installing Drew, confirm these prerequisites:

- You have `Administrator` permission in the target server.
- You have identified whether this environment is production or testing.
- You have one private channel reserved for initial setup.
- The Drew role is placed at the highest practical role position.
- Outbound connectivity is available for any optional external analysis services you enable.

If all checks pass, continue with [installation.md](installation.md).
